# Slay the Spire Deck Generator

Generates Slay the Spire decks using discrete mask-absorbing diffusion. The model is conditioned on character class and win outcome.

## Requirements

```
pip install torch tqdm matplotlib numpy
```

## Setup

Download `Monthly_2020_11.zip`, then run preprocessing to build the cleaned dataset and card vocabulary:

```python
from preprocess import init
init()
```

This writes `cleaned_runs.json` and `card_ids.json`.

## Configuration

All hyperparameters are in `config.py`:

| Parameter | Default | Description |
|---|---|---|
| `EPOCHS` | 25 | Number of training epochs |
| `BATCH_SIZE` | 128 | Training batch size |
| `LR` | 3e-4 | Learning rate |
| `MAX_DECK_SIZE` | 40 | Sequence length (deck slots) |
| `D_MODEL` | 128 | Transformer hidden dimension |
| `N_LAYERS` | 3 | Number of transformer layers |
| `N_HEADS` | 4 | Number of attention heads |
| `T` | 100 | Diffusion timesteps |
| `WIN_CONDITION` | True | Condition generation on winning |
| `WEIGHTED_SAMPLING` | True | Oversample winning decks during training |

## Training

```python
from training import train
train()
```

Training prints loss each epoch and appends it to `loss.txt`. Checkpoints are saved to `checkpoint.pt` after every epoch and training resumes automatically from the latest checkpoint. The best model by validation loss is saved to `best_model.pt`.

## Sampling

**Generate full decks from scratch:**

```python
from sample import sample_decks
sample_decks("best_model.pt", "IRONCLAD")
```

Valid characters: `IRONCLAD`, `THE_SILENT`, `DEFECT`, `WATCHER`.

**Complete a partial deck:**

```python
from sample import complete_deck

known_cards = [
    "Strike_R", "Strike_R", "Defend_R", "Defend_R", "Bash"
]

complete_deck("best_model.pt", "IRONCLAD", known_cards)
```

The known cards are held fixed throughout denoising and the remaining slots are generated by the model. The number of samples generated is controlled by `N_SAMPLES` in `config.py`.

## Visualisation

**Plot training loss curve:**

```python
from visualize import loss
loss()
```

Reads from `loss.txt` and saves the plot to `loss_curve.png`.

**Plot win/loss counts by character:**

```python
from visualize import runs
runs()
```
